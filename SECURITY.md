## Reportando uma Vulnerabilidade

Se você descobrir uma vulnerabilidade de segurança, por favor, **NÃO** abra uma issue pública. Em vez disso, envie um email para aureomanzano@icloud.com com os detalhes.

Por favor, inclua o seguinte na sua mensagem:

- Tipo de vulnerabilidade
- Componentes afetados
- Passos para reproduzir
- Impacto potencial
- Sugestões de correção (se houver)

Vamos responder dentro de 48 horas e trabalhar com você para resolver a questão antes de qualquer divulgação pública.

## Processo

1. Reporte a vulnerabilidade via email
2. Confirmação de recebimento (dentro de 48 horas)
3. Investigação e correção
4. Divulgação após correção (se aplicável)

## Reconhecimento

Agradecemos a todos que reportam vulnerabilidades de forma responsável. Seu esforço ajuda a manter o ZenWhats seguro para todos.

## Boas Práticas de Segurança

- Nunca compartilhe suas credenciais
- Use variáveis de ambiente para informações sensíveis
- Mantenha o projeto atualizado
- Revise regularmente as permissões de acesso
