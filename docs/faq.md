# FAQ - Perguntas Frequentes

## Perguntas Gerais

### O que é o ZenWhats?

ZenWhats é uma plataforma open source para integração e automação com WhatsApp, desenvolvida para facilitar a criação de soluções de comunicação automatizadas.

### Como posso contribuir?

Consulte nosso [Guia de Contribuição](../CONTRIBUTING.md) para mais informações sobre como contribuir com o projeto.

### O projeto é gratuito?

Sim! O ZenWhats é completamente open source e gratuito, licenciado sob a Licença MIT.

### Posso usar em produção?

O projeto está em desenvolvimento ativo. Use por sua conta e risco em ambientes de produção.

## Problemas Técnicos

### Como reportar um bug?

Abra uma [Issue](https://github.com/AureoManzanoJr/ZenWhats/issues) descrevendo o problema, incluindo:
- Descrição clara do bug
- Passos para reproduzir
- Comportamento esperado vs. atual
- Ambiente (OS, versão, etc.)

### Como solicitar uma nova funcionalidade?

Abra uma [Issue](https://github.com/AureoManzanoJr/ZenWhats/issues) com a tag "feature request" descrevendo a funcionalidade desejada.
