# Guia de Início Rápido

Bem-vindo ao ZenWhats! Este guia irá ajudá-lo a começar rapidamente.

## Instalação Rápida

### 1. Clone o Repositório

```bash
git clone https://github.com/AureoManzanoJr/ZenWhats.git
cd ZenWhats
```

### 2. Instale as Dependências

**Node.js:**
```bash
npm install
```

**Python:**
```bash
pip install -r requirements.txt
```

### 3. Configure o Ambiente

```bash
cp .env.example .env
```

Edite o arquivo `.env` com suas configurações.

### 4. Execute o Projeto

```bash
npm start
# ou
python main.py
```

## Primeiros Passos

Após a instalação, você pode começar a usar o ZenWhats seguindo os exemplos na pasta `examples/`.
